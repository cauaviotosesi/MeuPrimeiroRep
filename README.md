# Guia do Github
## 1° passo: criar um git
#### Criar um repositório no site do GuitHub e copiar seu URL
## 2° passo: criar uma pasta
#### Escolher ou criar uma pasta para começar o desenvolvimento do seu projeto.
## 3° passo: clonar seu repositório
#### abrir o terminal na sua pasta e executar o comando:
### "git clone [URL]"
#### Isso fará uma copia exata do seu repósitorio para sua maquina local!
## 3° passo: modificações
#### Para abrir o VSCODE de forma rapida execute os seguintes comandos no terminal:
### "cd NomeDoRepositório" & "code ."
#### Isso abrirá a pasta no VSCODE, onde você pode fazer suas alterações e lembre-se de SALVAR!
## 4° passo: salvar informações no seu GIT
#### Execute o comando:
### "git add ."
#### Para adicionar (selecionar) todos os arquivos, que serão atualizados.
## 5° passo: adicionar um commit (modificação)
#### Para marcar que você realizou uma modificação no github você precisa excutar o seguinte comando:
### "git commit -m ["Sua mesagem"]"
